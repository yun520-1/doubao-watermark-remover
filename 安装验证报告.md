## 🎯 完整安装和使用流程

### 步骤 1: 克隆仓库

```bash
git clone https://github.com/yun520-1/doubao-watermark-remover.git
cd doubao-watermark-remover
```

**验证**:
```bash
ls -la *.py
# 应看到:
# - final_perfect_v2_enhanced.py (v2 增强版)
# - final_perfect.py (标准版)
# - seedance_enhanced.py (自动检测版)
# - batch_final.py (批量处理)
```

### 步骤 2: 安装依赖

```bash
pip install -r requirements.txt
```

**安装内容**:
- opencv-python-headless (OpenCV 计算机视觉)
- numpy (数值计算)
- tqdm (进度条)
- scikit-image (图像处理)
- pillow (图像加载)

### 步骤 3: 使用水印去除

#### 方法 A: 使用 v2 增强版（推荐）

```bash
python3 final_perfect_v2_enhanced.py video.mp4
```

**输出**: `video_clean.mp4`

#### 方法 B: 指定输出文件

```bash
python3 final_perfect_v2_enhanced.py video.mp4 output.mp4
```

#### 方法 C: 批量处理

```bash
for file in *.mp4; do
    python3 final_perfect_v2_enhanced.py "$file"
done
```

## ❓ 常见问题

### Q: 安装失败？
**A**: 
```bash
# 升级 pip
python3 -m pip install --upgrade pip

# 重新安装
pip install -r requirements.txt --force-reinstall
```

### Q: FFmpeg 未安装？
**A**:
```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt-get install ffmpeg

# Windows
# 从 https://ffmpeg.org/download.html 下载
```

### Q: 视频模糊？
**A**: 使用 v2 增强版（已优化画质）
```bash
python3 final_perfect_v2_enhanced.py video.mp4
```

### Q: 处理速度慢？
**A**: 
- 使用标准版 `final_perfect.py`
- 或降低 FFmpeg preset 参数
