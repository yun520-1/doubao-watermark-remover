## ⏳ ClawHub 更新

**状态**: API 格式限制，需手动上传

**原因**: ClawHub API 需要特定的文件存储 ID (storageId)，该 ID 需要先上传文件到 ClawHub 存储才能获得。

**解决方案**:

### 方案 1: 通过 ClawHub 网站上传（推荐）

1. **访问项目页面**
   https://clawhub.ai/yun520-1/doubao-watermark-remover

2. **点击「编辑」或「上传新版本」**

3. **填写信息**:
   ```
   版本：2.0.0
   名称：豆包 AI 视频水印去除 (增强版 v2)
   描述：高清画质豆包 AI 视频水印去除工具，多算法融合检测、高级修复算法、画质增强
   Changelog: v2: 多算法融合检测、高级修复算法、画质增强、智能 CRF、音频优化
   标签：watermark, video, ai, doubao, v2, enhancement
   ```

4. **上传文件**:
   - 选择本地打包文件：`~/.jvs/.openclaw/workspace/qq-watermark-remover-v2.0.0.tar.gz`
   - 或上传单个文件

5. **点击发布**

### 方案 2: 使用提供的 Token 和脚本

已创建发布脚本 `publish-clawhub.js`，但需要 ClawHub API 支持。

```bash
cd ~/.jvs/.openclaw/workspace/qq-watermark-remover
node publish-clawhub.js
```
